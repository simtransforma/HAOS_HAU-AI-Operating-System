# Checklist — Rito Completo HAOS

## Gate 0 — Pronto para iniciar
- [ ] Tarefa classificada como operacional
- [ ] Escopo IN/OUT definido
- [ ] Papéis por etapa definidos
- [ ] Formato de reporte definido

## Gate 1 — Fases obrigatórias (sequência oficial)
- [ ] ABERTURA
- [ ] CONSELHO-Fase1
- [ ] CONSELHO-Fase2
- [ ] REPORT-SOLICITANTE
- [ ] EXECUÇÃO
- [ ] VALIDAÇÃO
- [ ] REGISTRO

## Gate 2 — Qualidade e prova
- [ ] Evidência por etapa registrada
- [ ] Status final sem bloqueio crítico
- [ ] Log e memória atualizados

## Gate 3 — Comunicação
- [ ] Atualizações com anti-silêncio atendido
- [ ] Resumo final enviado no formato padrão: [etapa][agente][ação][evidência][status/bloqueio]
