# Contributing to HAOS_HAU-AI-Operating-System

Obrigado por contribuir.

## Como contribuir
1. Abra uma issue descrevendo problema/ideia.
2. Crie branch: `feat/<tema>` ou `fix/<tema>`.
3. Envie PR com:
   - contexto
   - o que mudou
   - evidências/testes

## Padrão HAOS
Mudanças de processo devem respeitar o rito HAOS v2 documentado em `HAOS/`.

## Checklist mínimo no PR
- [ ] Docs atualizadas
- [ ] Sem quebrar regras de segurança
- [ ] Evidência de validação
