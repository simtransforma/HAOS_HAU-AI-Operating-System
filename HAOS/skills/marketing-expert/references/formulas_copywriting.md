# Fórmulas de Copywriting - Guia Completo

Referência detalhada das principais fórmulas de copywriting com exemplos para cursos de programação.

---

## AIDA

### Estrutura
**A**ttention (Atenção) - Headline que chama atenção
**I**nterest (Interesse) - Subtítulo que gera interesse
**D**esire (Desejo) - Corpo que cria desejo
**A**ction (Ação) - Call-to-action clara

### Quando Usar
- Landing pages
- Anúncios
- E-mails de vendas
- Posts de redes sociais

### Exemplo Completo - Curso de Python

**Attention:**
```
De Junior a Senior em 12 Semanas
(_mesmo que você esteja há 2 anos estagnado_)
```

**Interest:**
```
O programa comprovado que já transformou 200+ desenvolvedores
em profissionais de alto valor — com taxa de colocação de 94%
e salário médio de R$ 8.500
```

**Desire:**
```
Imagine daqui a 3 meses:

→ Trabalhando remoto para empresa americana
→ Com salário em dólar
→ Em projetos que realmente desafiam você

Não é sonho. É o que nossos alunos vivenciam todos os dias.

No programa você vai dominar:

✅ Arquitetura de Software Limpa
✅ Testes e Qualidade de Código
✅ DevOps e CI/CD
✅ Cloud Computing (AWS)
✅ Projetos Reais para Portfolio
✅ Preparação para Entrevistas

Tudo em 12 semanas. Com suporte individual.
Com comunidade ativa. Com garantia de resultado.

Alunos como Lucas:
→ De R$ 3.500 para R$ 12.000 em 8 meses
→ Trabalhando remoto para startup de San Francisco
→ "O melhor investimento da minha carreira"
```

**Action:**
```
Últimas 17 vagas para turma que começa segunda-feira.

[Quero minha vaga - Garantia 30 dias]

Preço sobe de R$ 997 para R$ 1497 à meia-noite de sexta.
```

## PAS (Problem-Agitate-Solution)

### Estrutura
**P**roblem (Problema) - Identifique a dor
**A**gitate (Agite) - Mostre consequências
**S**olution (Solução) - Apresente sua solução

### Quando Usar
- E-mails de conscientização
- Páginas de vendas
- Vídeos de vendas
- Posts para LinkedIn

### Exemplo - Curso Python

**Problem:**
```
Cansado de perder fins de semana debuggando código
que você mesmo escreveu há 2 semanas?

Você sabe o código "funciona" (afinal, roda sem dar erro),
mas você não tem ideia de COMO funciona.
```

**Agitate:**
```
Enquanto seus amigos estão na praia aproveitando o domingo,
você está preso no home office resolvendo problemas
que poderiam ter sido evitados.

E sabe o que é pior?

Em 6 meses, seu código vai ter 10x mais linhas.
10x mais complexo. 10x mais difícil de mantença.

E o problema não é falta de conhecimento técnico.
Você sabe syntax. Você sabe frameworks.

O problema é que ninguém te ensinou
a fazer BOM código.

Código limpo. Testável. Manutenível.

E a cada commit que você faz sem dominar isso,
você está cavando um buraco mais profundo.
```

**Solution:**
```
Domine as práticas que Top 1% dos desenvolvedores usam
para codar 3x mais rápido - com código que é fácil de
entender, testar e mantença.

Em 12 semanas você vai aprender:

→ Como estruturar código limpo (SOLID principles)
→ Como escrever testes que pegam bugs antes de produção
→ Como fazer refactoring sem medo de quebrar tudo
→ Como documentar código que outros devs entendem

Mesmo que você seja Júnior.
Mesmo que nunca tenha trabalhado em equipe.
Mesmo que seu código atual seja uma bagunça.

[Ver programa completo]

Próxima turma: Segunda-feira (17 vagas restantes)
```

## 4Ps (Picture, Promise, Prove, Push)

### Estrutura
**P**icture (Pinture) - Descreva situação ideal
**P**romise (Promessa) - Faça promessa do que vai entregar
**P**rove (Prove) - Dê provas de que funciona
**P**ush (Empurre) - Call-to-action

### Quando Usar
- Storytelling
- E-mails de transformação
- Páginas de histórico

### Exemplo

**Picture:**
```
Imagine acordar tomorrow, tomar seu café,
e abrir seu laptop para trabalhar no seu próprio ritmo.

Sem hora de chegada. Sem gerente perguntando onde você está.
Sem trânsito.

Imagine trabalhar em projetos que te desafiam,
com pessoas que você admira,
e sendo pago proporcionalmente ao seu valor.

Imagine poder viajar com sua família.
Trabalhar de qualquer lugar do mundo.
E ter liberdade financeira REAL.

```

**Promise:**
```
Este é o futuro que aguarda desenvolvedores
que dominam as habilidades certas.

E em 12 semanas, você pode estar lá.

Não é promessa vazia.
É método comprovado por 200+ alunos.
```

**Prove:**
```
Nossos números não mentem:

→ 94% taxa de colocação em até 6 meses
→ Salário médio: R$ 8.500
→ Aumento médio: 167%

Casos reais:

Lucas: De R$ 3.500 para R$ 12.000 em 8 meses
Mariana: Primeira vaga remota em 4 meses
Pedro: 5 ofertas de emprego após completar programa

```

**Push:**
```
Você está a 12 semanas de sua transformação.

[Inscrever-se agora - Vagas limitadas]

Garantia incondicional 30 dias.
Risco zero.
```

## 4Us (Useful, Urgent, Unique, Ultra-specific)

### Estrutura
**U**seful (Útil) - Deve ser útil para o público
**U**rgent (Urgente) - Deve criar urgência
**U**nique (Único) - Diferente de tudo
**U**ltra-specific (Ultra-específico) - Detalhado e preciso

### Quando Usar
- Validar headline
- Criar oferta
- Analisar concorrência

### Checklist para Validação

**Useful:**
- [ ] Resolve um problema real do público?
- [ ] Entrega valor claro?
- [ ] Melhora vida do cliente?

**Urgent:**
- [ ] Tem prazo/expiração?
- [ ] Cria necessidade de ação agora?
- [ ] Mostra consequência de não agir?

**Unique:**
- [ ] Diferente de concorrentes?
- [ ] Tem diferencial exclusivo?
- [ ] Proposta de valor única?

**Ultra-specific:**
- [ ] Tem números/detalhes específicos?
- [ ] Explica exatamente o que entrega?
- [ ] Remove ambiguidade?

### Exemplo de Validação

**Fraca:**
```
"Aprenda programação e conquiste sua liberdade"
```

**Problemas:**
- Não é urgente (não tem deadline)
- Não é única (todo curso promete isso)
- Não é específica (qual programação? que liberdade?)

**Forte:**
```
"Domine Python em 12 semanas e conquiste primeira vaga
remota com salário mínimo de R$ 8.000 — 94% dos nossos
alunos conseguem em até 6 meses ou seu dinheiro de volta"
```

**Por que funciona:**
- **Useful:** Promete vaga remota com salário específico
- **Urgent:** "12 semanas" + "próxima turma"
- **Unique:** Taxa de colocação 94% + garantia
- **Ultra-specific:** Python, 12 semanas, R$ 8.000, 94%

## Fórmula de 7 Passos (Gary Halbert)

### Estrutura Detalhada

**Passo 1: Crie Avatar Extremamente Específico**
```
Nome: Pedro
Idade: 24 anos
Ocupação: Desenvolvedor Junior há 2 anos
Salário: R$ 4.200
Dor: Faz 8 meses que não consegue passar de entrevista
Sonho: Trabalhar remoto para empresa americana
Medo: Ficar estagnado para sempre
```

**Passo 2: Descubra o Que Mais Desejam (Botão Emotional)**
```
Quando você aperta o botão emotional de Pedro:
- Ele quer RESPEITO (cansado de ser "menino" da equipe)
- Ele quer DINHEIRO (quer comprar apartamento)
- Ele quer LIBERDADE (quer viajar)
- Ele quer RECONHECIMENTO (quer ser Senior)

Botão MAIS poderoso: Liberdade + Dinheiro
```

**Passo 3: Mostre Veículo que Entrega o Desejo**
```
Veículo: Nosso programa de 12 semanas

O que o veículo promete:
→ Transformar Junior em Senior
→ Aumentar salário 2-3x
→ Conseguir vaga remota

Promessa clara: "12 semanas para trabalhar remoto em dólar"
```

**Passo 4: Preencha os Buracos que Impedem de Usar o Veículo**

**Buracos de Pedro:**
- "Não tenho tempo" → Programa é 10h/semana, no seu ritmo
- "Não tenho dinheiro" → 12x de R$ 97, garantia 30 dias
- "Não sou bom o bastante" → Começa do zero, passo a passo
- "Já tentei cursos e não funcionou" → Mentoria individual + projetos reais
- "Tenho medo de não conseguir" → 94% conseguem, você pode ser um deles

**Passo 5: Mostre Provas que Veículo Funciona**
```
→ 200+ alunos transformados
→ 94% taxa de colocação
→ Depoimentos em vídeo
→ Casos detalhados (Lucas, Mariana, Pedro)
→ Empresas que contrataram: Google, Amazon, Nubank
```

**Passo 6: Faça Oferta que Não Consegue Recusar**

**Oferta Base:**
- 12 semanas de programa
- Acesso ao curso completo
- Comunidade no Discord
- Certificado

**Bônus (Escassez):**
- [BÔNUS 1] Mentoria individual (valor: R$ 2.000)
- [BÔNUS 2] Revision de código (valor: R$ 1.500)
- [BÔNUS 3] Simulação de entrevista (valor: R$ 1.000)
- [BÔNUS 4] Template de portfolio (valor: R$ 500)
- [BÔNUS 5] Lista de vagas ocultas (valor: R$ 500)

**Valor Total: R$ 7.500 em bônus**
**Seu investimento: R$ 997**

**Passo 7: Dê Garantia que Remove Risco**
```
Teste o programa por 37 dias completos.

Se em qualquer momento você decidir que não é para você,
devolvemos 100% do seu dinheiro.

Sem perguntas. Sem letras miúdas.

Risco é TODO nosso. Você não tem nada a perder.
```

## Quando Usar Cada Fórmula

| Fórmula | Melhor Para | Exemplo de Uso |
|---------|-------------|----------------|
| **AIDA** | Landing pages, anúncios | Página de vendas completa |
| **PAS** | E-mails, vídeos de vendas | Conscientizar problema |
| **4Ps** | Storytelling, transformação | História de aluno |
| **4Us** | Validar copy, criar oferta | Analisar se headline é forte |
| **7 Passos** | Estruturação completa | Estruturar produto/oferta |

## Combinação de Fórmulas

**Você pode (e deve) combinar fórmulas:**

**Exemplo: E-mail usando PAS + Storytelling (4Ps)**

```
[PROBLEM]
Em 2019, eu estava onde você está.

Junior. Fazendo testes. Sem ser chamado.
Recebi 47 NÃOs até o primeiro SIM.

[AGITATE]
E o problema não era 'falta de conhecimento'.

Era falta de método.

Eu sabia fazer código que 'funciona'.
Mas empresas querem profissionais que fazem BOM código.

[SOLUTION]
E é exatamente isso que vou te ensinar.

[PICTURE]
Imagine daqui a 12 semanas:
→ Trabalhando remoto
→ Com salário em dólar
→ Em projetos que te desafiam

[PROMISE]
94% dos nossos alunos alcançam isso em até 6 meses.

[PROVE]
Lucas: De R$ 3.500 para R$ 12.000 em 8 meses
Mariana: Primeira vaga remoto em 4 meses

[PUSH]
Últimas 17 vagas. Turma começa segunda.

[Quero minha vaga]
```

## Dicas Finais

1. **Não fique preso em fórmulas** - Use como guia, não como regra rígida
2. **Teste variações** - A mesma oferta pode funcionar com estruturas diferentes
3. **Combine gatilhos mentais** - Fórmulas são estrutura; gatilhos são emoção
4. **Seja específico** - Sempre que possível, use números concretos
5. **Foque no resultado** - Cliente quer saber o que ganha, não o que é o produto
