# Template de E-mail Persuasivo

Use este template como estrutura base para e-mails de vendas.

---

## Estrutura do E-mail

```
[ASSUNTO]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[HOOK/GANCHO]
Primeiras 2-3 linhas que capturam atenção
(quebra de padrão, curiosidade, identificação)

[CONTEXTO/HISTÓRIA]
Desenvolve a narrativa
Conecta emocionalmente

[PROBLEMA/DOR]
Identifica a dor do leitor
Agita o problema

[SOLUÇÃO]
Apresenta sua solução
Como benefício, não como features

[PROVA]
Depoimentos, números, cases

[CTA]
Call-to-action clara e urgente

[ASSINATURA]
```

## Checklist para E-mail Persuasivo

Antes de enviar, verifique:

**Assunto:**
- [ ] Tem curiosidade ou benefício claro?
- [ ] Tem personalização (nome, "você")?
- [ ] Tem urgência ou escassez (se apropriado)?

**Corpo:**
- [ ] Primeiras 3 linhas capturam atenção?
- [ ] Conta história que conecta emocionalmente?
- [ ] Identifica dor do leitor?
- [ ] Agita o problema (mostra consequências)?
- [ ] Apresenta solução como benefício?
- [ ] Tem prova (números, depoimentos)?
- [ ] CTA é clara e específica?

**Formatação:**
- [ ] Parágrafos curtos (2-3 linhas)
- [ ] Uso de negrito para destacar
- [ ] Espaçamento para leitura fácil
- [ ] Links claros e destacados

**CTA:**
- [ ] Ação específica ("clique aqui", "inscreva-se")
- [ ] Urgência ("agora", "hoje", "últimas vagas")
- [ ] Repetido 2-3 vezes no e-mail
