# Template de Post para LinkedIn

Estrutura e exemplos de posts para LinkedIn que convertem.

---

## Estrutura de Post Viral no LinkedIn

```
1. HOOK (Primeiras 2 linhas) - Para de scroll
2. CONTEXTO - A história
3. CONFLITO - O problema
4. RESOLUÇÃO - A transformação
5. LIÇÃO APRENDIDA - O ensinamento
6. CTA - Chamada para ação
7. HASHTAGS
```

## Dicas para Posts no LinkedIn

### Formatação

**Use espaço entre parágrafos**
```
DENSIDADE RUIM:

Lorem ipsum dolor sit amet, consectetur adipiscing elit.
Sed do eiusmod tempor incididunt ut labore et dolore magna
aliqua. Ut enim ad minim veniam.

LEGÍVEL:

Lorem ipsum dolor sit amet.

Consectetur adipiscing elit.

Sed do eiusmod tempor incididunt.
```

**Use negrito estrategicamente**
```
Use para destacar:
→ Palavras-chave
→ Números/estatísticas
→ Benefícios
→ CTAs
```

**Use listas**
```
✅ Para conquistas
❌ Para erros/fracassos
→ Para passos/processos
🔥 Para engajamento
```

### Momento de Postar

**Melhores horários (Brasil):**
- Segunda a Sexta: 8h-9h, 12h-13h, 18h-19h
- Terça e Quinta são melhores dias
- Sábado/Domingo: evitar posts profissionais

### Engajamento

**Responda TODOS os comentários**
```
Nos primeiros 30 minutos:
→ Responda cada comentário
→ Faça perguntas de volta
→ Conecte-se com quem engajou
```

**Use first-degree comments**
```
Seu comentário no próprio post conta como engajamento
e ajuda algoritmo a distribuir mais.

Não abuse. Use para adicionar valor.
```

### Call-to-Actions que Funcionam

```
→ "Deixe um 🔥 se você quer o guia completo"
→ "Comenta 'SIM' que te envio o PDF"
→ "Salve para referência futura"
→ "Tag um amigo que precisa ver isso"
→ "Qual sua experiência? Comenta abaixo"
→ "Concorda ou discorda? Debate nos comentários"
```

### Hashtags

**Use 3-5 hashtags**
```
#CarreiraTech       (Ampla - muito volume)
#DesenvolvedorWeb   (Específica - menos volume)
#TechBrasil         (Local - público certo)
```

**Misture:**
- 2 amplas (#CarreiraTech, #Desenvolvimento)
- 2 específicas (#DeveloperJunior, #TrabalhoRemoto)
- 1 local (#TechBrasil)

```

[HOOK]
Me chega um LinkedIn message todo dia:

"Nossa, seu GitHub está incrível.
Como você construiu isso?"

Vou contar o segredo.
```

**[CONTEÚDO]**
```
6 meses atrás, meu GitHub era um deserto.

2 repos. Nada relevante.

O que mudou?

Eu parei de "fazer cursos"
e comecei a "construir projetos".

A diferença?

Cursos = Passivo. Assiste. Esquece.

Projetos = Ativo. Constrói. Mostra.

```

**[O QUE EU FIZ]**
```
Mês 1: Reconstruí projetos
→ Reescrevi app de ToDo usando Clean Architecture
→ Apliquei SOLID principles
→ Escrevi testes (80% coverage)

Mês 2-3: Expandi horizontes
→ Construí API REST completa
→ Criei frontend consumindo API
→ Deployei na AWS usando Docker

Mês 4-5: Contribuição
→ Encontrei 3 repos open source
→ Fiz contribuições significativas
→ Meu PR foi mergeado em projeto com 10k+ stars

Mês 6: Portfolio
→ Compilei tudo em portfolio
→ Documentei cada projeto
→ Criei demo videos de cada um
```

**[RESULTADO]**
```
Hoje:

→ 15 repos com projetos reais
→ 3 contribuições em projetos open source
→ 42 seguidores novos (incluindo recrutadores)
→ 5 pedidos de conexão de empresas em 1 semana
```

**[CTA]**
```
Seu GitHub é seu melhor currículo.

Construa como sua vida dependesse disso.
Porque, bem... depende.

Deixe um 🚀 se você quer o template
que uso para estruturar repos no GitHub.

#OpenSource #GitHub #CarreiraTech #Portfolio #Desenvolvedor
```

1. **Hook forte** - Me chega message TODO dia
