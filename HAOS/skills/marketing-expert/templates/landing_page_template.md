```
2. PROBLEMA + IDENTIFICAÇÃO
3. AGITAÇÃO (Consequências)
4. SOLUÇÃO (Apresentação do Curso)
5. PARA QUEM É (Target)
6. O QUE VOCÊ VAI APRENDER (Módulos)
7. PROVAS (Resultados, Depoimentos)
8. SOBRE O AUTOR/INSTRUTOR
9. OFERTA + BÔNUS
10. GARANTIA
11. FAQ
12. CTA FINAL (Escassez + Urgência)
```

## Seção 6: O Que Você Vai Aprender

```
12 SEMANAS. 6 MÓDULOS. TRANSFORMAÇÃO COMPLETA.
```

### Módulo 1: Fundamentos Sólidos (Semanas 1-2)
- Arquitetura limpa: SOLID principles
- Design Patterns essenciais
- Clean Code: como escrever código manutenível
- Refactoring sem medo

**Entregável:** 2 mini-projetos com código limpo

### Módulo 2: Testes e Qualidade (Semanas 3-4)
- TDD: Test-Driven Development
- Testes unitários, integração, e2e
- Code review: como revisar e ser revisado
- CI/CD: automação de qualidade

**Entregável:** Sistema testado com 80%+ coverage

### Módulo 3: API Development (Semanas 5-6)
- REST API: boas práticas
- GraphQL como alternativa
- Autenticação e autorização
- Documentação de API (OpenAPI/Swagger)

**Entregável:** API completa em produção

### Módulo 4: Frontend + Integration (Semanas 7-8)
- Integração Frontend-Backend
- State management
- Performance optimization
- Acessibilidade e SEO

**Entregável:** App full-stack funcionando

### Módulo 5: DevOps + Cloud (Semanas 9-10)
- Docker e containers
- Kubernetes básico
- AWS: deploy na nuvem
- Monitoring e logging

**Entregável:** App deployado na AWS com CI/CD

### Módulo 6: Carreira + Colocação (Semanas 11-12)
- Portfolio: como construir o matador
- LinkedIn optimizado para recrutadores
- Preparação para entrevistas técnicas
- Negociação salarial
- Procura ativa de vagas

**Entregável:** Sistema completo para conseguir vaga

## Checklist de Otimização

Antes de publicar, verifique:

**Headline:**
- [ ] Tem benefício claro?
- [ ] Tem especificidade (12 semanas, R$ 8.500)?
- [ ] Tem curiosidade?

**CTAs:**
- [ ] CTA principal visível "above the fold"?
- [ ] CTA repetido 5+ vezes na página?
- [ ] Contraste de cor suficiente?
- [ ] Texto do CTA é específico ("Quero minha vaga" não "Clique aqui")?

**Prova Social:**
- [ ] Tem números/estatísticas?
- [ ] Tem depoimentos em vídeo?
- [ ] Tem logos de empresas?

**Garantia:**
- [ ] Destacada visualmente?
- [ ] Prazo claro (37 dias)?
- [ ] "Risco zero" mencionado?

**Urgência/Escassez:**
- [ ] Vagas limitadas mencionadas?
- [ ] Data de início específica?
- [ ] Preço sobe em data específica?
- [ ] Countdown timer (se aplicável)?

**Velocidade:**
- [ ] Imagens otimizadas?
- [ ] Lazy loading ativado?
- [ ] CDN configurado?

**Mobile:**
- [ ] Responsivo em todos dispositivos?
- [ ] CTA clicável no mobile?
- [ ] Texto legível sem zoom?
