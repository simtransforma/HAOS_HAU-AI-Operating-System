# Recommended Technology Stack

## Frontend Stack
- **React 18** with TypeScript
- **Vite** (build tool)
- **TanStack Query** (server state)
- **Zustand** (global state)
- **Tailwind CSS** + **shadcn/ui**
- **Zod** (validation)
- **React Hook Form** (forms)

## Backend Stack
- **Node.js 18+**
- **Express.js** or **NestJS**
- **PostgreSQL** + **Prisma ORM**
- **JWT** + **NextAuth/Auth0**
- **Redis** (caching)
- **Winston** (logging)

## Testing Stack
- **Vitest** (unit tests)
- **Jest** (backend tests)
- **React Testing Library**
- **Playwright** (E2E)
- **MSW** (API mocking)

## DevOps Stack
- **Docker**
- **GitHub Actions**
- **AWS CDK** or **Terraform**
- **PostgreSQL** (database)
- **Redis** (cache)
