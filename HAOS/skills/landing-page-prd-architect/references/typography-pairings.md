# Typography Pairings

## Fontes para Headlines

| Fonte | Estilo | Uso |
|-------|--------|-----|
| Inter | Clean, moderno | Tech, SaaS, Startups |
| Poppins | Geométrico, friendly | Apps, Produtos |
| Montserrat | Elegante, versátil | Corporativo, Luxo |
| Plus Jakarta Sans | Moderno, legível | Tech, Design |
| Space Grotesk | Futurista | Tech, Crypto, AI |
| Outfit | Geométrico, suave | Lifestyle, Apps |
| Sora | Clean, contemporâneo | Startups, Tech |
| DM Sans | Minimalista | SaaS, Produtos digitais |
| Clash Display | Bold, impactante | Landing pages, Branding |
| Cabinet Grotesk | Moderno, distintivo | Design, Creative |

## Combinações Recomendadas

### Tech/SaaS
```
Headline: Space Grotesk (700)
Body: Inter (400, 500)
Accent: Inter (600)
```

### Startup/Moderno
```
Headline: Plus Jakarta Sans (700, 800)
Body: Inter (400)
Accent: Plus Jakarta Sans (600)
```

### Corporativo/Profissional
```
Headline: Montserrat (600, 700)
Body: Source Sans Pro (400)
Accent: Montserrat (500)
```

### Criativo/Design
```
Headline: Clash Display (600, 700)
Body: DM Sans (400)
Accent: Clash Display (500)
```

### Luxo/Premium
```
Headline: Cormorant Garamond (600)
Body: Lato (400)
Accent: Cormorant Garamond (500)
```

### Friendly/Lifestyle
```
Headline: Poppins (600, 700)
Body: Nunito (400)
Accent: Poppins (500)
```

### Minimalista
```
Headline: DM Sans (700)
Body: DM Sans (400)
Accent: DM Sans (500)
```

## Hierarquia Tipográfica

### Desktop
```
H1: 56-72px / line-height: 1.1
H2: 40-48px / line-height: 1.2
H3: 28-32px / line-height: 1.3
H4: 20-24px / line-height: 1.4
Body: 16-18px / line-height: 1.6
Small: 14px / line-height: 1.5
Caption: 12px / line-height: 1.4
```

### Mobile
```
H1: 36-44px / line-height: 1.15
H2: 28-32px / line-height: 1.2
H3: 22-24px / line-height: 1.3
H4: 18-20px / line-height: 1.4
Body: 16px / line-height: 1.6
Small: 14px / line-height: 1.5
```

## Weights Recomendados

```
Headlines: 600-800 (SemiBold a ExtraBold)
Subheadings: 500-600 (Medium a SemiBold)
Body: 400 (Regular)
Emphasis: 500-600 (Medium a SemiBold)
Captions: 400-500 (Regular a Medium)
```

## Letter Spacing

```
Headlines grandes: -0.02em a -0.03em (tracking negativo)
Subheadings: -0.01em a 0
Body text: 0 (normal)
Buttons/CTAs: 0.02em a 0.05em (tracking positivo)
All caps: 0.05em a 0.1em (tracking positivo)
```

## Google Fonts Import

```html
<!-- Tech/SaaS Stack -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Space+Grotesk:wght@500;600;700&display=swap" rel="stylesheet">

<!-- Startup/Moderno -->
<link href="https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@400;500;600;700;800&family=Inter:wght@400;500&display=swap" rel="stylesheet">

<!-- Corporativo -->
<link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@500;600;700&family=Source+Sans+Pro:wght@400;600&display=swap" rel="stylesheet">
```
