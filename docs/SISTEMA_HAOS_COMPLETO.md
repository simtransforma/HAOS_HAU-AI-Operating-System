# SISTEMA_HAOS_COMPLETO

> Documento explicativo. Em caso de conflito, prevalece `HAOS/`.

## Fluxo oficial único v2
ABERTURA -> CONSELHO-Fase1 -> REPORT-SOLICITANTE -> CONSELHO-Fase2 -> MEGA_BRAIN -> DIRETOR -> ESTRATEGISTA -> EXECUCAO -> VALIDACAO -> (se reprovado: CONSELHO_SE_REPROVADO [máx 3] -> MEGA_BRAIN) -> CONSELHO_Final_Aprovado -> ENTREGA -> REGISTRO

- CONSELHO-Fase1 = debate + perguntas ao solicitante
- REPORT-SOLICITANTE = consolidação da resposta do solicitante
- CONSELHO-Fase2 = direção após resposta do solicitante
